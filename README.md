## 使用说明

本文档会在更新 ``` main ``` 分支时自动部署；

正常使用仅需更新SUMMARY.md（目录配置）与./docs目录下相关内容

## 手动更新

不建议操作，可在需要更新的时候，执行如下命令：

```
sh script/gitbook.sh '更新说明'
```
